# Icons for 4 Minutes 33 Seconds

This directory should contain the following icon files:

## Required Icons

- **favicon-16x16.png** - Small favicon (16x16 pixels)
- **favicon-32x32.png** - Larger favicon (32x32 pixels)
- **icon-192x192.png** - Android home screen icon (192x192 pixels)
- **icon-512x512.png** - Android splash screen icon (512x512 pixels)
- **icon-192x192-maskable.png** - Maskable icon for adaptive icons (192x192 pixels)
- **icon-512x512-maskable.png** - Maskable icon for adaptive icons (512x512 pixels)

## Maskable Icons

Maskable icons are used by browsers that support adaptive icons (like Android 8+). They should have your design centered with at least 45px safe zone around the edges for 192x192 icons (or proportionally for larger sizes).

## Generating Icons

You can generate these icons using:

- Online tools like `https://www.favicon-generator.org/` or `https://icoconvert.com/`
- Icon design tools like Figma or Adobe XD
- Command line tools like ImageMagick: `convert icon.svg -size 192x192 icon-192x192.png`
